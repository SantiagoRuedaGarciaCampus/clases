Repositorio de clases en campus

De: Santiago Rueda Garcia, SantiagRuedaGarciaCampus

Tipo de archivos: js, css, html

Librerias: nodejs, reactjs

En cambio de tema se subira una carpeta con el nombre de nuevo tema y en esa se subira las clases de ese tema


En este repositorio se subiran tambien los trabajos de clase con una carpeta llamada 'ejercicio'
Lo trabajado en clase se ve en la carpeta 'my-app'


Se trabajara en el localhost:3000 en caso de ser necesario

tadas las clases vienen sin los node_modules, use npm i para instalarlos y usar correctamente los archivos
